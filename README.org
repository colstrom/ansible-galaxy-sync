#+TITLE: ansible-galaxy-sync
#+LATEX: \pagebreak

Commandline tool to re-import Galaxy roles.

* Installation

  #+BEGIN_SRC shell
    gem install ansible-galaxy-sync
  #+END_SRC

* Usage

  #+BEGIN_SRC shell
   ansible-galaxy-sync <role> [role ...]
  #+END_SRC
